PR 修改介面語言

語言包

【繁体】
【简体】
【日文】
【韓文】

下載好後，壓縮後將自己想要的語言放入Dictionaries資料夾內進行覆蓋即可!

資料夾路徑
C:\Program Files\Adobe\Adobe Premiere Pro CC 2018\Dictionaries

Windows

快捷鍵 ctrl + F12

Mac

快捷鍵 command + fn + F12

選取 Debug Database View



ApplicationLanguage

依自己喜好修改


  1. en_US 英文

  2. zh_TW 繁體

  3. zh_CN 簡體

使用雙語言的話，可以將以下↓

ApplicationLanguageBilingual

打勾 = true

關閉PR後重啟
